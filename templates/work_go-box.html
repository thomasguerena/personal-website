Go Box

front-end engineer

(2015)


I developed the Go Box mobile application. The project used Cordova (formally known as PhoneGap) and Google Polymer web components. The application allows users to find Go Box affiliated food carts, drop-offs, purchase additional Go Box tokens, and manage user accounts.


The idea was to use two high-level tools to quickly knockout a simple app. In reality, Polymer was too new (frequent code breaking updates) and too much of a pain to build for Cordova. After two months of part-time development with the talented Ruben Niculcea, struggling to get the two technologies to play nice, we spent an afternoon and replaced all the web components with standard HTML5. Voila. The experiment was fun while it lasted.


Ruben leads the development of all Go Box products. You can download the app for Android or iOS here, or visit the Go Box website for more information.
