iServeLife

font-end engineer / ui + ux / architect

(2015)


I was the architect and front-end developer for the web application iServeLife. The application, currently in closed beta, connects users with non-profit service opportunities relevant to their interests. The service provides users goals, service history, and certified transcripts which can be mailed directly to any recipient (e.g. university admissions). Users simply choose from a list of curated service opportunities and all further coordination is handled for them. For organization leaders, we provide user-progress tracking and organization-wide statistics.


The application was built using a RequireJS/Backbone front-end, a NodeJS Express server back-end, a SQL database, and a grunt build system.
